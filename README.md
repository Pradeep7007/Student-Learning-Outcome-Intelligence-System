# Student Learning Outcome Intelligence System (SLOIS)

This project consists of a React frontend and a Node.js/Express backend.

## Structure
- `client/`: React frontend
- `server/`: Node.js Express backend

## Backend Setup
1. Navigate to the server directory:
   ```bash
   cd server
   ```
2. Install dependencies (if not already done):
   ```bash
   npm install
   ```
3. Start the backend:
   ```bash
   npm run dev
   ```

## Frontend Setup
1. Navigate to the client directory:
   ```bash
   cd client
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the frontend:
   ```bash
   npm start
   ```

## Features
- **User Authentication**: Signup and Signin for Students, Staff, and Admins.
- **Auto-Uppercase**: Student/Staff names are automatically stored in uppercase in the database as per requirements.
